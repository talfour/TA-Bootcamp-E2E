# TA-Bootcamp-E2E

## Installation steps
### 1. Clone the repo
   ```sh
   git clone https://github.com/talfour/TA-Bootcamp-E2E.git
   ```

### 2. Install yarn packages
   ```sh
   yarn install
   ```
   ### OR
   ```sh
   npm install
   ```

### 3. To test the website
   ```sh
   npm run test
   ```
